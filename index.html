Connect Your Cloud Database

This web Admin Panel needs a database to read and update user account signups and crypto balances in real-time. We use free Firebase Firestore. Follow the instructions below to get connected in less than 2 minutes.

1

Create Firestore

Go to Firebase Console, create a free project, and enable Cloud Firestore (test mode).

2

Add Web App

In Project Settings, click "Add App" (Web). Copy the generated config values (Project ID, API Key, App ID).

3

Connect Here & App

Paste the keys below to connect. Also enter the exact same keys in your Android App's 9-dots menu sync dialog!

Disconnect Database?

Are you sure you want to unlink from the Firebase real-time database? This will:

  • Deactivate live sync with user apps
  • Clear connection state from session memory
  • Require database details to reconnect
